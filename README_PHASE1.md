## 🎯 快速開始

### 1️⃣ 啟動系統

```bash
cd C:\Users\josie\staff-scheduling-system
python -m http.server 8000
```

### 2️⃣ 開啟瀏覽器

```
http://localhost:8000/shift_management_system_sheets_full.html
```

### 3️⃣ 設定假日

```
設定頁籤 → 假日設定 → 選擇國家 → 重新載入假日
```

### 4️⃣ 開始使用！

```
排班檢視 → 點擊任一日期 → 自動開啟表單 → 儲存
```

## ⚡ 快速參考

### 常用操作

| 操作 | 方法 |
|------|------|
| 新增排班 | 點擊月曆日期 |
| 查看假日 | 紅色日期即為假日 |
| 更換國家 | 設定 → 假日設定 → 選擇國家 |
| 關閉假日 | 設定 → 取消勾選「顯示假日」 |

### 時間自動計算

```
09:00 → 11:00
14:00 → 16:00
23:00 → 01:00（跨日）
```

## ❓ 常見問題

**Q: 假日沒有顯示？**
```
A: 設定 → 確認「顯示假日」已勾選 → 重新載入假日
```

**Q: 點擊日期沒反應？**
```
A: 確認點擊日期方格 → 重新載入頁面（Ctrl+F5）
```

**Q: 時間沒有自動計算？**
```
A: 使用時間選擇器（點擊輸入框），不要手動輸入
```

**更多問題？** → 查看 [`PHASE1_USER_GUIDE.md`](PHASE1_USER_GUIDE.md)
