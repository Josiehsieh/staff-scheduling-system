## 💡 常見問題

### Q1: 地點欄位還是沒有變化？

```
診斷：
1. F12 → Console
2. 選擇事業單位
3. 查看是否有 📍 日誌

如果沒有日誌：
→ 重新整理網頁
→ 重新選擇單位

如果有錯誤訊息：
→ 檢查紅色錯誤
→ 重新登入系統
```

### Q2: 分散型單位沒有下拉選單？

```
原因：沒有設定分點

解決：
1. 到「🏢 事業單位」頁籤
2. 編輯該單位
3. 點擊「➕ 新增分點」
4. 填寫分點名稱和地址
5. 儲存
6. 回到排班頁面重新選擇
```

### Q3: 非分散型地址是空的？

```
原因：沒有設定地址

解決：
1. 到「🏢 事業單位」頁籤
2. 編輯該單位
3. 填寫地址欄位
4. 儲存

或者：
直接在排班時手動輸入地點 ✓
```
