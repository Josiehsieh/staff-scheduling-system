# 🎨 v6.1.3 更新 - 移除顏色設定

## 變更內容

### ❌ 移除

- 顏色選擇器（調色板）
- 顏色選擇功能
- 相關 CSS 和 JavaScript

### ✅ 保留

- 固定使用顏色：**#667eea**（紫藍色）
- 所有核心功能正常運作

## 現在的使用方式

### 新增排班

1. 點擊「➕ 新增排班」
2. 填寫資料
3. **不需要選擇顏色**（自動使用預設色）
4. 儲存

### 視覺效果

- 🎨 所有排班使用統一的紫藍色
- 📅 月曆顯示更簡潔統一
- ✨ 更專業的外觀
