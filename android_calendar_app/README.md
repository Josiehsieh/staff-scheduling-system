# 醫療行事曆應用程式

一個功能完整的安卓行事曆應用程式，專為醫療人員設計，包含排班同步、專案管理、月經週期記錄、記帳等功能。

## 主要功能

### 📅 行事曆功能
- **多日曆支援**: 個人行事曆、Google行事曆、醫療排班系統
- **10種顏色選擇**: 與排班系統不同的顏色方案，避免衝突
- **農曆顯示**: 支援農曆日期和節日顯示
- **事件管理**: 創建、編輯、刪除事件
- **重複事件**: 支援週期性事件設定
- **Widget支援**: 桌面小工具顯示今日事件

### 🏥 排班系統同步
- **HTML解析**: 自動解析排班系統網頁資料
- **即時同步**: 定期同步排班資料
- **衝突檢測**: 避免與個人事件時間衝突
- **視覺區分**: 排班事件使用專用紅色標示

### 📋 專案管理
- **專案追蹤**: 創建和管理專案
- **進度監控**: 視覺化進度條顯示
- **優先級設定**: 低、中、高、緊急四個等級
- **狀態管理**: 未開始、進行中、暫停、已完成、已取消
- **預算追蹤**: 專案預算設定和監控

### 🌸 月經週期記錄
- **週期追蹤**: 記錄月經開始和結束日期
- **智能預測**: 基於歷史數據預測下次週期
- **症狀記錄**: 記錄相關症狀和不適
- **流量強度**: 輕、中、重三級記錄
- **統計分析**: 平均週期長度和經期長度

### 💰 記帳功能
- **收支記錄**: 收入和支出分類記錄
- **類別管理**: 自定義收支類別
- **預算設定**: 月度、週度預算管理
- **統計圖表**: 收支趨勢和分類統計
- **收據拍照**: 支援收據圖片記錄

### 🎨 顏色管理
- **10種主要顏色**: 珊瑚紅、青綠色、天藍色、薄荷綠、金黃色、梅紅色、海藍綠、檸檬黃、薰衣草紫、淺藍色
- **專案顏色**: 10種專案專用顏色
- **記帳顏色**: 10種記帳類別顏色
- **智能對比**: 自動計算文字對比色

### 📱 Widget功能
- **今日事件**: 顯示今日所有事件
- **快速操作**: 直接從Widget添加事件
- **自定義樣式**: 多種Widget樣式選擇
- **即時更新**: 自動同步最新資料

## 技術架構

### 前端技術
- **Jetpack Compose**: 現代化UI框架
- **Material Design 3**: 最新設計語言
- **Navigation Compose**: 單活動導航
- **ViewModel**: MVVM架構模式

### 後端技術
- **Room Database**: 本地資料庫
- **DataStore**: 設定資料儲存
- **WorkManager**: 背景任務處理
- **Hilt**: 依賴注入

### 外部整合
- **Google Calendar API**: Google行事曆同步
- **Supabase**: 雲端資料庫
- **Firebase**: 推播通知和分析
- **JSoup**: HTML解析

## 安裝說明

### 系統需求
- Android 7.0 (API 24) 或更高版本
- 至少 100MB 可用儲存空間
- 網路連線（用於同步功能）

### 安裝步驟
1. 下載 APK 檔案
2. 啟用「未知來源」應用程式安裝
3. 安裝應用程式
4. 授予必要權限（日曆、儲存空間、網路）
5. 設定Google帳號（可選）
6. 設定排班系統URL（可選）

### 權限說明
- **日曆權限**: 讀取和寫入行事曆事件
- **儲存空間**: 儲存應用程式資料和收據圖片
- **網路**: 同步Google行事曆和排班系統
- **相機**: 拍攝收據照片（記帳功能）

## 使用指南

### 首次設定
1. **顏色選擇**: 在設定中選擇偏好的顏色主題
2. **同步設定**: 設定Google帳號和排班系統URL
3. **月經週期**: 輸入基本週期資訊
4. **記帳類別**: 設定常用的收支類別

### 日常使用
1. **查看行事曆**: 主畫面顯示當月行事曆
2. **添加事件**: 點擊日期或使用+按鈕
3. **專案管理**: 切換到專案標籤頁
4. **週期記錄**: 在月經週期頁面記錄
5. **記帳**: 在記帳頁面記錄收支

### 同步功能
- **自動同步**: 每小時自動同步一次
- **手動同步**: 下拉刷新或點擊同步按鈕
- **衝突解決**: 系統會提示時間衝突

## 開發資訊

### 開發環境
- Android Studio Hedgehog | 2023.1.1
- Kotlin 1.9.20
- Gradle 8.5
- Compose BOM 2024.02.00

### 專案結構
```
app/src/main/java/com/medical/calendar/
├── data/
│   ├── local/          # Room資料庫
│   ├── model/          # 資料模型
│   ├── remote/         # 網路服務
│   └── repository/     # 資料庫操作
├── ui/
│   ├── calendar/       # 行事曆畫面
│   ├── event/          # 事件管理
│   ├── projects/       # 專案管理
│   ├── menstrual/      # 月經週期
│   ├── finance/        # 記帳功能
│   ├── settings/       # 設定畫面
│   └── navigation/     # 導航
├── util/               # 工具類
└── widget/             # Widget功能
```

### 建置說明
```bash
# 克隆專案
git clone [repository-url]

# 開啟Android Studio
# 等待Gradle同步完成

# 建置APK
./gradlew assembleDebug

# 安裝到裝置
./gradlew installDebug
```
