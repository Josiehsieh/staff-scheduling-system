# ⚡ 事業單位管理 - 2分鐘快速開始

## 🚀 立即開始

### 步驟 1：前往事業單位頁籤

```
登入系統 → 點擊「🏢 事業單位」頁籤
```

### 步驟 2：填寫表單

```
事業單位名稱 *：台北診所

是否為分散型：否

地址：台北市信義區

醫生姓名：
王醫師
李醫師

護理師姓名：
陳護理師
林護理師

一月時數：
醫生：160
護理師：180
其他：0

... 填寫其他月份
```

### 步驟 3：儲存

```
點擊「💾 儲存事業單位」
```

### 步驟 4：完成！

```
✅ 事業單位「台北診所」新增成功！
✅ 自動儲存到 Google Sheets
✅ 排班時可以選擇了
```

---

## 📊 Google Sheets 設定

**自動建立**：系統會自動在您的 Google Sheets 中建立「**事業單位詳細**」工作表

**欄位結構**（共 42 欄）：
- A: 名稱
- B: 分散型（是/否）
- C: 地址
- D: 醫生名單
- E: 護理師名單
- F: 其他人員名單
- G-I: 一月時數（醫/護/其他）
- J-L: 二月時數
- ... 每月3欄
- AK-AM: 十二月時數

---

## 💡 快速提示

### 時數填寫技巧

**全職醫生（每週 40 小時）**：
```
一月：176 小時
二月：160 小時
三月：176 小時
...
```

**兼職護理師（每週 20 小時）**：
```
一月：88 小時
二月：80 小時
三月：88 小時
...
```

### 快速填入相同時數

如果每月時數相同，可以：
1. 填好一月的時數
2. 複製數字
3. 貼上到其他月份

---

## ✏️ 編輯 & 刪除

### 編輯

```
1. 在列表中找到事業單位
2. 點擊「✏️ 編輯」
3. 修改資料
4. 點擊「💾 儲存事業單位」
```

### 刪除

```
1. 在列表中找到事業單位
2. 點擊「🗑️ 刪除」
3. 確認刪除
```

---

## ⚠️ 重要提醒

1. ✅ 事業單位名稱必須填寫
2. ✅ 時數可以填 0 或留空
3. ✅ 資料會立即同步到 Google Sheets
4. ⚠️ 刪除操作無法復原
