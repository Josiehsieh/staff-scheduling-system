## 🔍 常見問題

### Q1: 還是沒有顯示人員？

```
診斷：
1. F12 → Console
2. 選擇事業單位
3. 查看日誌

如果看到空陣列 []：
→ 該單位沒有設定人員名單
→ 到事業單位頁籤設定

如果看到錯誤訊息：
→ 該單位可能不在「事業單位詳細」表中
→ 需要重新建立或編輯
```

### Q2: 時數顯示 0h？

```
原因：該月沒有設定配額

解決：
1. 到「🏢 事業單位」頁籤
2. 編輯單位
3. 填寫該月的時數
4. 儲存
```

### Q3: 顯示「該單位無人員資料」？

```
解決：
1. 到「🏢 事業單位」頁籤
2. 點擊該單位的「✏️ 編輯」
3. 填寫人員名單：
   醫生名單：王醫師, 李醫師
   護理師名單：林護理師
   其他人員：張助理
4. 儲存
5. 回到排班頁面重新選擇
```
