# 📅 行事曆功能快速開始

**v6.0.0 新功能** - 上傳排班到 Google 行事曆  
**閱讀時間**：1分鐘

---

## ⚡ 3步驟開始使用

### 1️⃣ 啟用 Calendar API

```
前往：https://console.cloud.google.com/
  → 選擇您的專案
  → API 和服務
  → + 啟用 API 和服務
  → 搜尋「Google Calendar API」
  → 啟用
```

### 2️⃣ 重新登入授權

```
系統中：
  → 點擊「登出」
  → 重新登入
  → 授權行事曆權限（新增）✓
```

### 3️⃣ 開始上傳

```
排班管理頁籤：
  → 單個上傳：點擊「📅 行事曆」按鈕
  → 批量上傳：點擊「📅 批量上傳行事曆」
```

---

## 📱 結果

**在 Google 行事曆中：**
- ✅ 顯示所有排班
- ⏰ 30分鐘前提醒
- ⏰ 1天前提醒
- 📱 手機自動同步

---

## 🎯 使用範例

**排班：**
```
日期：2025-11-15
時間：09:00 - 11:00
單位：台北診所
人員：王醫師, 李醫師
```

**上傳後在行事曆：**
```
標題：台北診所 - 排班
時間：2025-11-15 09:00 - 11:00
提醒：30分鐘前 + 1天前
```

## ❓ 常見問題

**Q: 權限不足？**
```
A: 確認已啟用 Calendar API + 重新登入
```

**Q: 找不到事件？**
```
A: 檢查「主要」行事曆 + 確認日期
```

**Q: 會重複上傳嗎？**
```
A: 會！每次點擊都會建立新事件
   建議：先測試單個，再批量上傳
```
